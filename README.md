# MNIST CNN Classifier (PyTorch)

This project is a clean and simple **Convolutional Neural Network (CNN)** built using **PyTorch** to classify handwritten digits from the MNIST dataset.

## 🧠 Model Architecture
- 2 Convolution Layers
- ReLU Activation
- Fully Connected Layers
- Output: 10 digit classes (0–9)

## 📁 Files
- `model.py` → CNN architecture  
- `train.py` → Model training  
- `test.py` → Model evaluation  
- `saved_model/` → Trained model weights  
- `requirements.txt` → Install dependencies
